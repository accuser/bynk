Bynk by Example is a problem-first tour of the language: each page starts with a
thing you want to do and shows the real code that does it. It comes in two tiers —
_snippets_, single-file language primitives you can open and type-check in the
browser playground, and _projects_, complete example apps that each compile to a
deployable Cloudflare Worker. Read them in any order; every example type-checks as
shown.
